# Stock Market Risk Analytics — Power BI

An interactive Power BI dashboard analysing stock price movements, trading trends,
market sentiment, and risk metrics to support data-driven investment decisions.

## Highlights
- Price-movement and trading-trend analysis
- Sentiment overlay on market performance
- Risk analytics and key financial indicators
- Interactive, filterable Power BI report

## Tech
Power BI · DAX · data modelling · financial analytics

## Files
- `DV_Stock_Price_Movements_Trading_Trends_Sentiment_Risk_Analytics.pbix` — Power BI report
  (open with Power BI Desktop)
